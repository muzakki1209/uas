Jawaban UAS no.1

Website adalah kumpulan halaman web yang dapat diakses melalui internet. Cara kerja sebuah website melibatkan beberapa komponen utama, yaitu:

  1. Kode HTML/CSS/JavaScript: Halaman web dibuat dengan menggunakan bahasa pemrograman seperti HTML (Hypertext Markup Language) untuk struktur, CSS (Cascading Style Sheets) untuk tata letak dan desain, dan JavaScript untuk interaktivitas.
  2. Browser (Pengunjung): Pengguna mengakses website menggunakan perangkat lunak browser seperti Chrome, Firefox, atau Safari. Browser mengambil kode HTML, CSS, dan JavaScript dari server dan menampilkan halaman web tersebut di layar pengguna.
  3. Web Server: Web server adalah perangkat keras atau perangkat lunak yang menyimpan, mengelola, dan mengirimkan halaman web kepada pengguna. Web server menerima permintaan dari browser pengguna, memprosesnya, dan mengirimkan kembali halaman web yang diminta.
  4. Domain: Domain adalah alamat unik yang digunakan untuk mengidentifikasi sebuah website di internet. Misalnya, www.nama-website.com. Domain digunakan oleh pengguna untuk mengakses website tanpa harus mengingat alamat IP server tempat website tersebut disimpan.
  5. Hosting: Hosting adalah layanan penyimpanan dan pemeliharaan file website di server. Ketika Anda memiliki website, Anda perlu menyimpan file-file tersebut di server yang terhubung ke internet agar bisa diakses oleh pengguna. Hosting menyediakan ruang penyimpanan ini dan memungkinkan website Anda online.

    6. Hubungan antara Web Server, Domain, dan Hosting:

    • Web Server dan Hosting: Web server adalah tempat di mana file website disimpan dan diakses oleh pengguna. Hosting adalah layanan yang menyediakan server tempat Anda dapat menyimpan file-file tersebut agar dapat diakses oleh orang lain.
    • Domain dan Web Server/Hosting: Domain digunakan sebagai alamat yang mudah diingat oleh manusia untuk mengakses website. Ketika seseorang memasukkan domain (misalnya www.nama-website.com) ke dalam browser, domain tersebut akan dihubungkan dengan alamat IP server tempat website disimpan. Ini memungkinkan browser untuk mengirim permintaan ke server yang sesuai dan mengambil halaman web yang diinginkan.
